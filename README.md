## Interesting Projects

Here are some projects that I think are interesting.

### Just Hexagons

Just Hexagons is a open-source ultimate hexagon utility Unity package.
The main feature is a `Hexagon` definition which is inspired by the Vector structs and is used similarly.

It is made with C# in Unity.

Here is the repo: [Hexagons](https://github.com/Rodak123/Hexagons)\
Here is the package: [Just Hexagons](https://assetstore.unity.com/packages/tools/utilities/just-hexagons-345388?srsltid=AfmBOor2pO8EPnOpb7NrGrI1-Kw-RqfNnx24WVxXrWIjGqr6vt-svBhj)

### Modelab

Modelab is an in development content management website that I helped to make.\

The frontend is made in TypeScipt with React + Vite and Tailwind CSS.
The backend is made in PHP (no frameworks).

Here is the repo: [Modelab](https://github.com/DanielHaas21/Modelab) and [Modelab-api](https://github.com/DanielHaas21/Modelab-api)

### Termitaire

Termitaire is a open-source klondike solitaire made for the terminal.
The entire game is built around being text-based and controlled with only a keyboard.

It is made in Java.

Here is the repo: [Termitaire](https://github.com/Rodak123/Termitaire)

### p5-webpack-kit

This kit is a under development JavaScript library which easily allows to use the framework p5.js inside node.js using webpack.
It provides easy-to-use wrappers for p5.js functions. 

It is made in JavaScript.

Here is the repo: [p5-webpack-kit](https://github.com/Rodak123/p5-webpack-kit)\
Here is the npm package: [p5-webpack-kit](https://www.npmjs.com/package/p5-webpack-kit)
